# Google Clone

This is a Google clone project that replicates the basic functionality of the Google search engine. It allows users to search for information using a search bar and displays search results based on fake data APIs.

## Features

- Search functionality: Users can type queries into the search bar and receive search results.
- Fake data APIs: The project uses fake data APIs to simulate search functionality.The fake data used is comments.
- Responsive design: The application is designed to work seamlessly on both desktop and mobile devices.

## Technologies Used

- React: The project is built using the React JavaScript library.
- React Router: Used for handling client-side routing within the application.
- Axios: Used for making HTTP requests to the fake data APIs.
- Tailwind CSS: Used for styling the user interface components.

## Installation

1. Clone the repository:

git clone https://github.com/digambarJail/Google-Clone.git


2. Navigate to the project directory:

cd Google-Clone


3. Install dependencies:

npm install


4. Start the development server:

npm run dev


5. Open your browser and visit `http://localhost:5173` to view the application.

## Usage

- Enter a search query into the search bar.
- Press Enter or click the search button.
- View the search results displayed on the page.
